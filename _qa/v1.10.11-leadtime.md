## Honest scope (gaps documented in changelog + motor.md)

- ONNX session options patched via env, NOT via libpiper.h. Patching `piper_create` to take an `OrtSessionOptions` builder requires forking piper1-gpl. Env path covers `intra_op_num_threads` + `inter_op_num_threads`; `graph_optimization_level` + arena flags have no env equivalent and are deferred.
- Dither knob declared but not wired — `zaudio.Engine.Config` doesn't expose `dither_mode`. The env value is parsed and logged so a future patch can wire it via custom `data_callback` over `ma_data_converter`.
- `-3 dB` headroom is global, not per-utterance adaptive. Per-call peak normalisation deferred to v1.11 postfx track.
