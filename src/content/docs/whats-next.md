## TL;DR

**The entire v1.6 → v1.10 slate shipped on 2026-06-03**, plus v1.1 → v1.5 earlier the same day. Thirteen + five = eighteen milestones, all measured.

- **v1.6 Voice cloning ship-it** — XTTS-v2 sidecar live; real Gabriel voice; 5 install blockers fixed
- **v1.7 Streaming text input** — `agent-tts stream` + `say_stream` MCP tool + incremental chunker
- **v1.8 SSML & prosody** — `<emphasis>` / `<break>` / `<prosody>` / `<say-as>` for say + piper; parse < 0.2 µs
- **v1.9 Web playground** — Astro widget scaffold; WASM synth deferred to v1.9.1
- **v1.10 Menubar UI** — SwiftUI status item + queue + Skip/Clear + voice picker; ducking + Linux GTK deferred to v1.10.1

See the [Changelog](/changelog/) for measurements + honest scope per version. The next slate (v1.11+) is unscheduled.
